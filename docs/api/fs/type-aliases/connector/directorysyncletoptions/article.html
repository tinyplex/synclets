DirectorySyncletOptions

The DirectorySyncletOptions type specifies configuration for creating a directory-backed Synclet.

(
  DirectoryDataConnectorOptions<Depth> &
  DirectoryMetaConnectorOptions<Depth> &
  {
    transport: Transport;
    implementations?: SyncletImplementations<Depth>;
  } &
  SyncletOptions
)
TypeDescription
transportTransport

The transport property specifies the Transport instance for synchronization.

implementations?SyncletImplementations<Depth>

The implementations property optionally specifies custom conflict resolution implementations.
