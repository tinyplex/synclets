FileConnectorsOptions

The FileConnectorsOptions type specifies configuration for both data and meta connectors when using createFileConnectors.

{
  dataFile?: string;
  metaFile?: string;
}
TypeDescription
dataFile?string

The dataFile property specifies the file path for data storage.

metaFile?string

The metaFile property specifies the file path for metadata storage.

Since

v0.0.5
