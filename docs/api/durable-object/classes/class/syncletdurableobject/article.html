SyncletDurableObject

Extend this class and optionally implement the createDataConnector and createMetaConnector methods to define how your Durable Object stores data and metadata. The class automatically manages the Synclet lifecycle internally - you don't need to initialize or access the synclet directly.

Optionally implement getSyncletImplementations or getSyncletOptions to configure custom implementations or other options.

Example

class MySyncletDurableObject extends SyncletDurableObject {
  createDataConnector() {
    return createMemoryDataConnector({depth: 3});
  }
  createMetaConnector() {
    return createMemoryMetaConnector({depth: 3});
  }
  getSyncletOptions() {
    return {logLevel: 'debug'};
  }
}
