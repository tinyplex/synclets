DurableObjectSqliteDataConnectorOptions

The DurableObjectSqliteDataConnectorOptions type describes the options for creating a data connector that uses Cloudflare Durable Object's built-in SQLite storage.

{sqlStorage: SqlStorage} & DatabaseDataConnectorOptions<Depth>

Example

const dataConnector = createDurableObjectSqliteDataConnector({
  depth: 3,
  sqlStorage: this.ctx.storage.sql,
  dataTable: 'data',
  addressColumn: 'address',
  atomColumn: 'atom',
});
