createMemoryConnectors

The createMemoryConnectors function creates both an in-memory DataConnector and MetaConnector together in a single call, returning them as a tuple.

createMemoryConnectors<Depth>(depth: Depth): Connectors<Depth, DataConnector<Depth>, MetaConnector<Depth>>
TypeDescription
depthDepth

The tree depth the Synclet will operate at.

returnsConnectors<Depth, DataConnector<Depth>, MetaConnector<Depth>>

A tuple of [DataConnector, MetaConnector].

This is the simplest way to create a fully functional Synclet with no persistence, making it ideal for testing, prototyping, or scenarios where data should not survive process restarts. Both connectors store their data in JavaScript Maps for fast access.

These are the default connectors used by createSynclet when no connectors are explicitly provided.
