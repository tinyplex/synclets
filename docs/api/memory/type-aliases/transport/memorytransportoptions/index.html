MemoryTransportOptions

The MemoryTransportOptions type specifies configuration for creating an in-memory Transport.

(
  {poolId?: string} &
  TransportOptions
)
TypeDescription
poolId?string

The poolId property optionally specifies a pool identifier for isolating groups of Synclets.
