createSessionStorageConnectors

The createSessionStorageConnectors function creates both a sessionStorage-backed DataConnector and MetaConnector together in a single call, returning them as a tuple.

createSessionStorageConnectors<Depth>(
  depth: Depth,
  dataStorageName: string,
  metaStorageName: string,
): Connectors<Depth, SessionStorageDataConnector<Depth>, SessionStorageMetaConnector<Depth>>
TypeDescription
depthDepth

The tree depth the Synclet will operate at.

dataStorageNamestring

The sessionStorage key for data storage.

metaStorageNamestring

The sessionStorage key for metadata storage.

returnsConnectors<Depth, SessionStorageDataConnector<Depth>, SessionStorageMetaConnector<Depth>>

A tuple of [DataConnector, MetaConnector].

This is the recommended approach for browser-based Synclets where both data and metadata should be ephemeral and scoped to a single tab. The function provides a simplified API compared to creating the connectors separately.

The data and metadata are stored under separate sessionStorage keys, allowing you to organize your storage namespace.
