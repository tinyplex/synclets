Timestamp

The Timestamp type represents a serialized Hybrid Logical Clock (HLC) value stored as a string. Timestamps are used for conflict resolution during sync, with later timestamps winning over earlier ones. The HLC format combines physical time with a logical counter to provide total ordering even when system clocks are not perfectly synchronized.
