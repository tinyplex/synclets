SyncletComponents

The SyncletComponents type bundles the optional connector and transport instances supplied to createSynclet.

{
  dataConnector?: DataConnectorType;
  metaConnector?: MetaConnectorType;
  transport?: Transport | Transport[];
}
TypeDescription
dataConnector?DataConnectorType

The dataConnector property supplies the DataConnector instance to use.

metaConnector?MetaConnectorType

The metaConnector property supplies the MetaConnector instance to use.

transport?Transport | Transport[]

The transport property provides one or more Transport instances.

v0.0.0
